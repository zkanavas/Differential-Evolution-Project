What Happens if the Mutated Vector Maxes Out?

Shown below are the weight values assigned by the differential evolution algorithm for each iteration. There were a total of 81 iterations for this example. The maximum value reached was 9.99783 for the Euclidean distance during iteration #56. In the next iteration, #57, the value of the Euclidean distance weight was 5.40038228.

When the mutation vector maxes out as it does here, the algorithm proceeds as normal, mutating the current best vector with two random vectors. The algorithm may retain that high weight value, but only if it results in a high fitness score - namely, a high accuracy.
